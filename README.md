# Automate Folder Organizer Script

## Description

A Python automation script that organizes files into folders based on file type.

## Features

* Detects file types automatically
* Creates folders automatically
* Moves files into categories
* Handles duplicate filenames safely
* Organizes Images, Documents, Audio, Videos, and Archives

## Technologies Used

* Python
* OS Module
* Shutil Module

## How to Run

1. Open the project folder in VS Code.
2. Run `folder_organizer.py`.
3. Enter the folder path.
4. Files will be automatically organized.
